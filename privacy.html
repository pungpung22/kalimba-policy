개인정보처리방침

Kalimba - 칼림바 시뮬레이터

1. 개인정보 수집 여부

본 앱(Kalimba)은 사용자의 개인정보를 수집하지 않습니다.

Kalimba는 회원가입, 로그인, 서버 연동 기능이 없는 오프라인 앱입니다. 따라서 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

2. 기기 내 저장 데이터

앱 사용 시 다음 데이터가 사용자의 기기에만 저장됩니다:

  • 앱 설정 (배경색, 건반 테마, 볼륨 등)
  • 사용자가 녹음한 연주 데이터
  • 사용자가 저장한 악보 데이터

이 데이터는 외부 서버로 전송되지 않으며, 앱을 삭제하면 함께 삭제됩니다.

3. 제3자 제공

본 앱은 사용자 데이터를 제3자에게 제공하지 않습니다.

4. 광고 및 분석 도구

본 앱은 Google AdMob을 통해 광고를 표시합니다.

AdMob은 광고 제공을 위해 다음 정보를 수집할 수 있습니다:

  • 기기 식별자 (광고 ID)
  • IP 주소
  • 앱 사용 정보

자세한 내용은 Google 개인정보처리방침을 참조하세요.

5. 데이터 삭제

기기에 저장된 모든 데이터는 다음 방법으로 삭제할 수 있습니다:

  • 앱 삭제: 기기에서 앱을 삭제하면 모든 데이터가 삭제됩니다.
  • 앱 데이터 초기화: 기기 설정에서 앱 데이터를 초기화할 수 있습니다.
