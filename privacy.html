Kalimba - 칼림바 시뮬레이터

📊 Google Play 데이터 보안 요약

Google Play Console의 "데이터 보안(Data Safety)" 양식과 1:1로 매핑되는 요약입니다.

  • 수집: 광고 ID, IP 주소, 앱 사용 이벤트 (모두 Google AdMob에 의한 자동 수집)
  • 공유: 광고 ID·IP·앱 이벤트 → Google AdMob (광고 목적)
  • 수집 안 함: 이름·이메일·전화번호 등 개인 식별 정보 일절 수집하지 않음 (회원가입/로그인 없는 오프라인 앱)
  • 데이터 판매(Sell): 없음
  • 전송 구간 암호화: HTTPS/TLS 1.2 이상
  • 데이터 삭제 요청: 앱 삭제 또는 기기 설정에서 앱 데이터 초기화
  • 대상 연령: 전체 이용가 (악기 시뮬레이터, Google Play Families 정책 비대상)

1. 개인정보 수집 여부

본 앱(Kalimba)은 사용자의 개인정보를 수집하지 않습니다.

Kalimba는 회원가입, 로그인, 서버 연동 기능이 없는 오프라인 앱입니다. 따라서 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

2. 기기 내 저장 데이터

앱 사용 시 다음 데이터가 사용자의 기기에만 저장됩니다:

  • 앱 설정 (배경색, 건반 테마, 볼륨 등)
  • 사용자가 녹음한 연주 데이터
  • 사용자가 저장한 악보 데이터

이 데이터는 외부 서버로 전송되지 않으며, 앱을 삭제하면 함께 삭제됩니다.

수집하지 않는 정보 (최소 수집 원칙)

  • 개인 식별 정보: 이름, 이메일, 전화번호, 생년월일, 주소
  • 민감정보: 주민등록번호, 신용카드, 계좌번호, 생체정보, 건강·종교·정치 정보
  • 통신 데이터: SMS, 통화 기록, 연락처(주소록)
  • 하드웨어 식별자: IMEI, MAC 주소, 시리얼 번호 (Google의 재설정 가능한 광고 ID만 AdMob이 사용)
  • 위치 정보: GPS, Wi-Fi 기반 정밀 위치 (AdMob의 IP 기반 대략적 지역만 해당)
  • 기기 내 파일: 갤러리 사진/동영상 스캔, 타 앱 데이터, 설치된 앱 목록
  • 실시간 입력: 마이크 녹음(앱 내 연주 녹음은 기기에만 저장), 카메라, 키 입력, 화면 캡처

3. 제3자 제공

본 앱은 사용자 데이터를 제3자에게 제공하지 않습니다.

4. 광고 및 분석 도구

본 앱은 Google AdMob을 통해 광고를 표시합니다.

AdMob은 광고 제공을 위해 다음 정보를 수집할 수 있습니다:

  • 기기 식별자 (광고 ID)
  • IP 주소
  • 앱 사용 정보

이용자는 기기 설정의 "광고 ID 재설정/삭제"(Android) 또는 "추적 허용" 해제(iOS)를 통해 맞춤형 광고를 제한할 수 있습니다.

AdMob 광고 파트너

AdMob은 Google이 승인한 제3자 광고 네트워크(예: Meta Audience Network, AppLovin, Unity Ads 등)를 통해 입찰·중개할 수 있습니다.

  • Google 인증 광고 구매자 및 공급 파트너 목록
  • Google 파트너 사이트 및 앱의 정보 수집 방법
  • Google 광고 정책
  • Google 개인정보처리방침

5. 데이터 삭제 (Google Play 요구사항 준수)

본 앱은 서버에 개인정보를 저장하지 않으므로 별도의 "계정 삭제" 개념이 존재하지 않습니다. 모든 데이터는 기기 내에만 저장되며, 아래 방법으로 즉시 완전 삭제됩니다:

  • 앱 삭제: 기기에서 앱을 삭제하면 모든 데이터가 삭제됩니다.
  • 앱 데이터 초기화: 기기 설정 → 앱 → Kalimba → 저장공간 → 데이터 삭제.
  • 광고 ID 재설정: 기기 설정 → Google → 광고 → 광고 ID 삭제/재설정으로 AdMob에 저장된 광고 이력을 리셋할 수 있습니다.

참고: Google Play 「계정 삭제」 요구사항은 계정 기반 앱을 대상으로 합니다. 본 앱은 계정이 없으므로 해당 요구사항이 원칙적으로 적용되지 않으나, 동등한 수준의 삭제 경로를 위와 같이 제공합니다.
